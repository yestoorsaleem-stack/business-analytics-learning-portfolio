# Pandas

This folder contains Python practice notebooks related to Pandas from my Programming for Business course.

## Topics Included

- Loading datasets
- Inspecting data
- Viewing rows and columns
- Selecting single and multiple columns
- Using `loc` and `iloc`
- Checking missing values
- Basic descriptive statistics
- Basic data exploration

## Purpose

The purpose of this section is to show how Pandas can be used to load, inspect, clean, and explore structured business datasets in Python.
