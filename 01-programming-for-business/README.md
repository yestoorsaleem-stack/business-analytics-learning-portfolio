The course focused on building a foundation in Python programming, computational thinking, problem solving, and applying programming concepts to business and data analysis tasks.

- While loops and for loops
- Nested loops
- Functions
- Local and global variables
- Files and exceptions
- Lists and tuples
- Dictionaries and sets
- NumPy
- Pandas
- Exploratory Data Analysis
- Matplotlib
- Seaborn
